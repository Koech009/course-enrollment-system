# Course Enrollment System

A simple Python project demonstrating sequence operations using a real-world course enrollment scenario.

## Features

- Display enrolled courses
- Add new courses
- Remove courses
- Filter courses using list comprehensions
- Process courses lazily using generator expressions

## Concepts Covered

- Python sequences (lists)
- CRUD operations
- Iteration
- List comprehensions
- Generator expressions

## How to Run

```bash
python course.py
```
